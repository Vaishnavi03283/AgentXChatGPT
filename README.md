# AgentXChatGPT

AgentXChatGPT is a chatit application that utilizes OpenAI's GPT-3.5-turbo model to provide conversational AI capabilities. It is designed to be user-friendly and can be integrated into various platforms for interactive communication.

# How to run AgentXChatGPT 
 ## 1. Clone Repository 
    '''bash
    git clone https://github.com/Vaishnavi03283/AgentXChatGPT.git
    ...
    
 ## 2. Navigate to the project Directory : 
    cd AgentXChatGPT

## 3. Create a virtual environment (optional) but recommended:
    conda create -n agentxchatgpt python=3.11 -y

## 4. Activate  the virtual environment :
    conda activate agentxchatgpt

## 5. Install the required dependencies:
    pip install -r requirements.txt

## 6. Run the application:
    python app.py
